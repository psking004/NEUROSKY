# NeuroSky Frontend

Production-grade UI for the AI-Powered Drone Traffic Management System.

## Features

- Real-time drone tracking via WebSocket
- Collision alert visualization
- Modern dark theme dashboard
- Custom animated drone markers
- Responsive layout

## Quick Start

```bash
npm install
npm run dev
```

Access at http://localhost:5173

## Tech Stack

- React 18
- Vite 5
- Leaflet + React-Leaflet
- Socket.IO Client
